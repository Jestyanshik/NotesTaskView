# NotesTaskView Shortcuts

## Overlay

- `Win + Shift + Tab` - show or hide the overlay.
- `F5` - refresh the current folder.
- `Esc` - close a menu, dialog, search, preview, clear selection, go back, or hide the overlay depending on the current mode.

## Navigation

- `Arrow keys` - move focus between cards.
- `Enter` - open the focused item.
- `Enter hold` - start keyboard drag.
- `Delete` - delete the selected item after confirmation.
- `Right Shift` or `Menu` - open the custom overlay context menu.

## Selection

- `Ctrl + Click` - toggle item selection.
- `Alt + Click` - toggle item selection.
- `Shift + Click` - select a range.
- `Ctrl/Alt + Arrow` - move focus without resetting selection.
- `Shift + Arrow` - extend selection.
- `Ctrl/Alt + Enter/Space` - toggle the focused item.

## Drag

- `Left mouse drag` - move or reorder items.
- `Drag onto folder` - move selected items into that folder.
- `Drag to the upper header while inside a folder` - navigate one level up.
- `Group drag` - drag multiple selected items together.

## Preview

- `Middle mouse hold` - show preview.
- Release middle mouse - hide preview.

## Search

- `Ctrl + F` in the list - global search by title and content.
- `Ctrl + F` in the editor - search inside the current file.
- `Enter` in file search - next match.
- `Shift + Enter` in file search - previous match.
- `Esc` - close search.

## Editor

- `Ctrl + S` - save the current note.
- `Esc` - return from the editor using the unsaved-changes flow.
